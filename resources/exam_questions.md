# Past Exam Questions — Maritime Routing & Scheduling

## Context (shared across questions)

Both question sets relate to ship routing and scheduling in the context of **tramp shipping**,
where vessels transport bulk cargo for customers under both long-term contracts and spot market arrangements.

## Question Set 2 — Star Carriers

**Scenario:**
You have applied for a job as an optimisation expert at the shipping company Star Carriers,
which has formulated its route planning problem and subsequently developed a solution method
based on **Dantzig-Wolfe decomposition**. The resulting path flow model is shown below.
Star Carriers wants to test your optimisation knowledge before offering you the job, and
therefore asks the following questions.

### The Path Flow Model (Star Carriers)

$$
\max \sum_{v \in V} \sum_{r \in R_v} (P_{vr} - C_{vr}) x_{vr}
$$

Subject to:

$$
\sum_{v \in V} \sum_{r \in R_v} A_{ivr} x_{vr} = 1 \qquad \forall i \in N_C
$$

$$
\sum_{v \in V} \sum_{r \in R_v} A_{ivr} x_{vr} \leq 1 \qquad \forall i \in N_O
$$

$$
\sum_{r \in R_v} x_{vr} = 1 \qquad \forall v \in V
$$

$$
x_{vr} \in \{0, 1\} \qquad \forall v \in V,\ r \in R_v
$$

**Notation:**

| Symbol | Description |
|---|---|
| $V$ | Set of vessels |
| $R_v$ | Set of feasible routes for vessel $v$ |
| $P_{vr}$ | Revenue generated by vessel $v$ sailing route $r$ |
| $C_{vr}$ | Cost incurred by vessel $v$ sailing route $r$ |
| $x_{vr}$ | Binary variable: 1 if vessel $v$ sails route $r$, 0 otherwise |
| $A_{ivr}$ | Binary parameter: 1 if cargo $i$ is carried by vessel $v$ on route $r$, 0 otherwise |
| $N_C$ | Set of mandatory contracted cargoes |
| $N_O$ | Set of optional spot market cargoes |

### Part (a)

Is this a problem within **tramp** or **industrial** shipping? Justify your answer.
Also show briefly how the model can be changed to represent the *opposite* of these two
(i.e., either from tramp to industrial shipping, or from industrial to tramp shipping).

### Part (b)

Based on the model above, state whether Star Carriers can have accounted for the following
in their solution method *(answers must be justified)*:

**I.** Speed optimisation
(where the optimal speed on each sailing leg is determined)

**II.** Flexible cargo quantities
(where the quantity of each cargo can be decided within a given interval)

**III.** Split loads
(where a cargo is allowed to be split and transported by more than one vessel)
